# 👥 Добавление FivelySino бота в группу Telegram

## 📋 Что нужно сделать

### Шаг 1: Настройка бота через BotFather

1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Отправь команду `/mybots`
3. Выбери своего бота **FivelySino** (или как ты его назвал)
4. Нажми **Bot Settings**
5. Нажми **Allow Groups?**
6. Выбери **Turn groups on** ✅

Теперь бот может быть добавлен в группы!

### Шаг 2: Дополнительные настройки (опционально)

Для лучшей работы в группах:

1. В BotFather выбери своего бота
2. **Bot Settings** → **Group Privacy**
3. Выбери **Turn off** (чтобы бот видел все сообщения) или оставь **Turn on** (бот видит только команды)

**Рекомендация:** Оставь **Group Privacy ON**, так как боту не нужно читать все сообщения в группе.

### Шаг 3: Добавление бота в группу

1. Открой свою группу в Telegram
2. Нажми на название группы вверху
3. Нажми **Add Members** (Добавить участников)
4. Найди своего бота по имени
5. Добавь бота в группу

ИЛИ

1. Перейди к боту в личных сообщениях
2. Нажми меню (три точки) → **Add to Group**
3. Выбери нужную группу

### Шаг 4: Назначение прав (опционально)

Если хочешь, чтобы бот мог удалять сообщения или имел другие права:

1. Открой группу
2. Нажми на название группы
3. **Administrators** → **Add Administrator**
4. Выбери бота
5. Настрой нужные права

**Важно:** Для работы игры права администратора НЕ нужны! Бот работает без них.

## 🎮 Как играть в группе

После добавления бота в группу:

1. Любой участник группы может написать `/start@имя_вашего_бота`
2. Бот отправит сообщение с кнопкой
3. Нажми **🎰 Крутить барабаны**
4. Играй!

### Команды в группе:

```
/start@имя_бота - начать игру
/balance@имя_бота - посмотреть баланс
/addcoins@имя_бота - добавить монеты
```

**Совет:** Замени `имя_бота` на username твоего бота (например, `FivelySino_bot`)

### ✅ Что работает:

- Все видят результаты игры

## 🔧 Устранение проблем

### Бот не отвечает в группе
1. Убедись, что бот добавлен в группу
2. Проверь, что включена опция **Allow Groups** в BotFather
3. Убедись, что бот не забанен/заблокирован в группе
4. Убедись, что бот запущен (скрипт работает на твоём компьютере)

### Бот не видит команды
- Используй команды с упоминанием бота: `/start@твой_бот`
- Или дай боту права администратора (хотя это необязательно)

### В группе несколько ботов
- Всегда указывай username бота после команды: `/start@FivelySino_bot`

## 🎊 Советы для групповой игры

1. **Создай отдельный чат** для казино - так удобнее играть
2. **Установи правила** - например, ограничение на количество игр в день
3. **Сделай турнир** - кто наберёт больше монет за неделю
4. **Добавь призы** - победитель получает что-то реальное 🎁

## 📊 Идеи для группы

### 🏆 Таблица лидеров
Периодически спрашивай у игроков их баланс командой `/balance` и веди таблицу вручную.

### 🎲 Турниры
Все начинают с одинакового баланса (используй `/addcoins`), играют определённое время, побеждает тот, у кого больше всего монет.

### 💰 Общий банк
Можно вести общий банк группы - когда кто-то выигрывает много, группа получает "налог" 😄
